1. [Write a program to create an ArrayList, ask the user for `n`, take `n` integers from the user, store them in the list, and print the ArrayList.](#problem1)
2. [Write a program to create an ArrayList, take user input to fill it, and then insert a new element at a user-given index.](#problem2)
3. [Ask the user for a value → remove that value.  Then ask for an index → remove the element at that index.](#problem3)
4. Ask the user for an index and a new value → update (replace) the element.

## Solutions:
#### problem1
```java
package arrayListSolutions;

import java.util.Scanner;
import java.util.ArrayList;

public class AddElements {

	public static void main(String[] args) {
		Scanner input = new Scanner(System.in);
		
		System.out.print("number of input: ");
		int size = input.nextInt();
		
		ArrayList<Integer> al = new ArrayList<Integer>();
		
		System.out.print("Enter " + size + " integers separated by a whitespace: ");
		for(int i=0; i<size; i++ ) {
			al.add(input.nextInt());
		}
		
		System.out.println("Array Lists are: " + al);
		input.close();
	}

}
```

#### problem2
```java
package arrayListSolutions;

import java.util.Scanner;
import java.util.ArrayList;

public class InsertElementsAtIndex {

	public static void main(String[] args) {
		Scanner input = new Scanner(System.in);
		
		System.out.print("number of input: ");
		int size = input.nextInt();
		
		ArrayList<Integer> al = new ArrayList<Integer>();
		
		System.out.print("Enter " + size + " integers separated by a whitespace: ");
		for(int i=0; i<size; i++)  {
			al.add(input.nextInt());
		}
		
		System.out.println("Array List: " + al);
		
		System.out.print("Enter the index no. where you want to insert a value: ");
		int index = input.nextInt();
		
		System.out.print("What value you want to insert at index no.'" + index + "': ");
		int value = input.nextInt();
		
		al.add(index, value);
		System.out.println("Updated Array List: " + al);
		
		input.close();
	}

}
```

#### problem3
```java
package arrayListSolutions;

import java.util.Scanner;
import java.util.ArrayList;

public class RemoveElementAtIndex {

	public static void main(String[] args) {
		Scanner input = new Scanner(System.in);
		
		System.out.print("number of input: ");
		int size = input.nextInt();
		
		ArrayList<Integer> al = new ArrayList<Integer>();
		
		System.out.print("Enter " + size + " integers separated by a whitespace: ");
		for(int i=0; i<size; i++)  {
			al.add(input.nextInt());
		}
		
		System.out.println("Array List: " + al);
		
		System.out.print("Enter the index no. where you want to delete a value: ");
		int index = input.nextInt();
		
		al.remove(index);
		System.out.print("After deletion at index '" + index + "', new updated Array List: " + al);
		
		input.close();
	}

}
```
